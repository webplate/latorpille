# A phonebook for the community

## Dependencies:

- python3
- Flask
- htmldom
- tinydb

##Try it

	$ cd the/repository/clone
	virtualenv -p python3 venv
	. venv/bin/activate
	pip install Flask htmldom tinydb
	
Run debug with:

    python server.py
    
Then visit:

    localhost:5000

server.py is also a wsgi app.
